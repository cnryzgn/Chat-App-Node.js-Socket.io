* Express   --version 4.18.1
* Socket.io --version 4.5.1

## Setup

```bash

# Before start please make sure to have Git And Node

# Clone
$ git clone https://github.com/cnryzgn/Chat-App-Node.js-Socket.io.git

# Open Directory
$ cd Chat-App-Node.js-Socket.io

# Install
$ npm install

# Run
$ npm start

```
